# FYG-AFS-API

Requeriments
=======
-Node.js 8.11.1 or stable

-NPM 5.6.0 or stable

-SQL Server 2017

-Have this repo cloned


Instalation
===========
In the shell, enter to the repo and install dependencies:

```bash
$ npm install
```


Run project
===========
```bash
$ npm run debug
```

Open localhost in port 3000
